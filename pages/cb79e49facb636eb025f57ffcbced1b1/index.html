第十章 精液检验

一、学习目标

掌握 精液理学检查和显微镜检查。
熟悉 精液标本的采集、精液化学检查和计算机辅助精子分析系统。
了解 精液显微镜检查各项指标的质量控制和方法评价。

二、重点和难点内容

（一）精液标本采集

精液由精子和精浆组成。精液标本采集以手淫法为宜精液标本采集对禁欲时间室温控制、容器和保温温度的要求等质量控制是精液检验分析前质量控制的重要内容

（二）理学检查

包括精液外观、量、凝固及液化、黏稠度和酸碱度的检查。精液透明度和颜色、精液量（精液减少、无精液症和精液增多症）凝固及液化时间等理学改变可反映男性泌尿系统的病理情况和评价男性生殖功能

（三）显微镜检查

采用普通光学显微镜观察未染色精液标本的有形成分和染色后的精子形态。

  1. 精子活动率：是指显微镜下直接观察活动精子所占精子总数的百分率。精子活动率减低是导致男性不育的重要因素。
  2. 精子活动力：是指精子前向运动的能力。WHO将精子活动力分为3级。精子活动力力低下，难以抵达输卵管或无力与卵子结合而不能完成受精过程。
  3. 精子存活率：采用活精子所占比例表示。精子存活率减低是导致不育的重要原因之一。
  4. 精子凝集试验：是指活动的精子相互黏附在一起。精子凝集提示可能为免疫因素引起的不育。
  5. 精子计数：包括精子浓度和精子总数。睾丸或输精管病变可导致精子数量减少。
  6. 精子形态：在液化后的精液涂片干燥后进行染色。正常精液中畸形精子不超过20％。畸形精子增加可影响男性生育能力。
  7. 精子低渗肿胀试验：是观察精子在低渗溶液中的变化，以检测精子膜的完整性。

（四）化学检查

通过精液化学成分检查，反映附属性腺的分泌功能，对男性不育症的诊疗及病因分析有重要临床意义。

包括：酸性磷酸酶、乳酸脱氢酶-X、中性a-葡萄糖苷酶、精子顶体精氨酸酰胺酶、枸橼酸、果糖、锌等检查。

（五）计算机辅助精子分析系统（CASA）

该系统采用计算机分析技术和图像处理技术相结合，通过摄像机或录像与显微镜连接，确定和跟踪单个精子的活动，根据设定的精子运动的移位、精子大小和灰度及精子运动的有关参数，对采集到的图像进行动态处理分析。CASA既可定量分析精子浓度、精子活动力、精子活动率，又可分析精子运动速度和运动轨迹等特征。


三、习题

名词解释

  1. 精液液化时间：指新排出的精液由胶冻状转变为流动状液体的时间，一般＜60分钟。刚离体精液由于精囊腺分泌的凝固酶作用而呈胶冻状随后在前列腺分泌的纤溶酶作用下液化。液化不全或完全不液化可抑制精子活动力导致不育。
  2. 精子活动力：指精子向前运动的能力，是一项直接反映精子质量的指标。WHO将其分为3级:前向运动、非前向运动和无运动。精子活动力减低常见于精索静脉曲张、生殖系统非特异性感染等，是导致不育症的原因之一。
  3. 精子低渗肿胀试验：观察精子在低渗溶液中的变化情况，检测精子膜的完整性。本试验可作为精子膜功能及完整性的评估指标可预测精子潜在的受精能力。
  4. 精液黏稠度：指完全液化后的黏度。可用直接玻棒法或 Pasteur滴管法测定精液液化不全或完全不液化可阻抑精子活动力，影响受孕。

简答题

1.简述精液检查的临床应用。

精液检查的临床应用包括：

  1. 评价男性生育功能，供不育症的诊断和疗效观察依据。
  2. 男性生殖系统疾病的辅助诊断及疗效判断。
  3. 输精管结扎术后的效果观察，术后6周后精液内应无精子存在。
  4. 为体外受精和精子库筛选优质精子。
  5. 为法医学鉴定提供依据

2.试述精液标本采集的注意事项。

精液标本采集的注意事项包括：

  1. 检查前应向患者解释精液标本采集和送检方法、禁欲时间（2～5天）、排尿等的说明。标本采集前应至少禁欲48小时，但不超过7天。一般情况下，30岁以下禁欲2～3天，30～40岁禁欲～5天，40岁以上禁欲5～7天，需连续2～3次检查的，2次之间一般应间隔1～2周，但不超过3周。
  2. 标本采集室最好在实验室附近，室温控制在20～35℃。
  3. 推荐用手淫法采集精液标本。应收集排出的全部精液。
  4. 标本容器应洁净、干燥，不能用安全套作为容器，以免影响精子活力
  5. 采集的精液若同时用于微生物培养，必须无菌操作。
  6. 采集后需在1小时内送检。冬季标本应于20～40℃保温送检。

3.试述精子活动率与存活率的关系。

精子活动率与存活率的关系如下：

  1. 两者定义及检查方法不同，其异常都易导致不育
  2. 活动率是指活动精子占精子总数的百分率，而存活率是利用染料对液化精液进行染色，观察不着色的活精子占精子总数的百分率。活精子不一定有活动能力，活动的精子则肯定是活精子。
  3. 引起活动率下降的原因有:精索静脉曲张、内分泌异常、全身性疾病及生殖道感染如精囊病变、前列腺炎症、免疫性抗体产生等，此外还见于过度饮酒、吸烟、药物（环磷酰胺）、慢性疾病等。
  4. 引起精子存活率下降的原因有精索静脉曲张、生精功能障碍、附属性腺器官炎症、隐睾、睾丸局部温度过高等。

4.试述计算机辅助精子分析系统的检测原理和方法评价。

计算机辅助精子分析系统（CASA）是20世纪80年代发展的新技术。

其检测原理是：采用计算机分析技术和图像处理技术相结合，通过摄像机或录像与显微镜连接，确定和跟踪单个精子的活动，根据设定的精子运动的移位、精子大小和灰度及精子运动的有关参数，对采集到的图像进行动态处理分析并打印结果。

方法评价：

  1. 优点：CASA具有高效客观、高精度的特点，但价格较昂贵。
  2. 影响因素：CASA系统识别精子的准确性受精液中细胞和非细胞成分的影响CASA测定的是单个精子的运动参数，缺乏对精子群体的了解。
  3. 局限性：CASA对检测精子浓度有一定局限性，在（20～50）×10L的范围内检测结果较理想。
  4. 目前，WHO仍推荐使用显微镜直接检测精子浓度和精子活动率。
